# Fourteen Hours Apart

Long-form English novel project for Amazon KDP / Kindle.

## Project Identity

- Working title: `Fourteen Hours Apart`
- Language: English
- Target platform: Amazon KDP / Kindle
- Genre: second-chance romance, contemporary romantic fiction, mature cross-cultural romance
- Core settings: present-day Seoul and New York, with flashbacks to a fictional New Jersey state university
- Commercial positioning: an English-language second-chance long-distance romance, not a memoir and not a translation of any prior Korean book

## Core Question

Nathan Kim and Elaine Park lost each other once because they were too young to choose love. Twenty years later, they may lose each other again because their lives are too complete.

The story is not simply about whether they love each other. They do. The real question is:

Can two people who lost each other twenty years ago change the lives they built without each other?

## Repository Guide

- `00_project_bible/`: story rules, characters, timeline, locations, themes, style, and privacy boundaries
- `01_raw_episodes/`: cleaned source episodes derived from author-provided memories
- `02_scene_cards/`: fictionalized scene planning cards generated from raw episodes
- `03_outline/`: act structure and evolving chapter outline
- `04_draft/`: chapter drafts, created only after enough episodes and outline work exist
- `05_revision/`: revision notes, continuity checks, and structural passes
- `06_market_package/`: KDP description, subtitle options, comp titles, and positioning

## Working Method

1. Add one raw episode at a time to `01_raw_episodes/`.
2. Convert each raw episode into a scene card in `02_scene_cards/`.
3. Update outline placement candidates in `03_outline/`.
4. Track privacy risks before using real emotional material in fiction.
5. Draft chapters only after the project bible, enough scene cards, and chapter outline support the chapter.

The project folder is the memory. Do not rely on chat memory for continuity.
